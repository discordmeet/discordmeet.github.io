What is this?

Discord.Meet was built with the intention of enabling Discord users to meet new people in one-on-one conversations, instead of only through single-interest servers. Joining a big Server on something you really like can be fun but also scary to some. That's why this service tries to serve an alternative method.

The core part of this service is a Bot which matches people up and acts as a middleman, forwarding messages from one to the other and back in order to allow for anonymity.

How to join?

There are two seperate methods to join:
The first, recommended way requires a little initial setup but grants complete anonymity towards your chatpartners.
The second method is far simpler, but cannot fully guarantee anonymity towards your chatpartners.
Click on whichever method you feel comfortable with!

The anonymous way (Recommended):

Step 1:

Create a new Discord server.
Give it any name, image or region you like (defaults are fine).
Modify the server however you like, the Bot will only add a textchannel.

Step 2:

Click the button below to invite the bot to your server.
You may have to sign into Discord in your browser.
Discord will ask you which server the Bot should be added to, choose the server you just created and click OK.

Get started

There are a couple steps you can take to make your expierience on Discord.Meet more enjoyable.
If you just wanna get going skip to point 3.

1. Set your character card

The character card can be used to give your partner some info about yourself right after matching.
Read more on what that looks like here.

2. Set your matching preferences

You can customize what people you'll be matched with via some parameters, like what gender your partner should have. Read more on Matching Preferences here.

3. Meet some people

Now that you're all set up you can start meeting new people! To get going simply type !search.
If you decide you wanna meet someone else during and active chat, type !next.

What are "Character cards"?

"Character cards" are special embeds containing information set by a user to be shown to their partner on a match.
You can set the following fields:

  • Nickname
  • Profile Image
  • Description
  • Gender
  • Interests

Matching Preferences

Not implemented in Beta!

Who you'll be matched with depends on a couple of parameters.
You can set:
  • What Gender your partner should have
  • What country your partner should be from
  • Wether or not you're looking for an NSFW chat
  • What interests your partner should have in common


Interests are a bit special, as they're a weak criterium, meaning if noone with matching interests is waiting that parameter is ignored. The other three are strong criteria, meaning if noone with matching ones is in queue you won't be matched.

Command Reference

A list of all available commands. Please note the use of quotation marks in the commands!
If your input to the command has multiple words that should count as one object, please wrap it in quotation marks! (see !setinterests example)

Navigation

  • !search - Start searching for a chatpartner
  • !leave - Leave current conversation/Stop searching
  • !next - Leave current conversation and immediately start searching for a new one
  • !state - Query your current state. Either 'Idle', 'Searching' or 'in Conversation'

Settings

  • !setname "NICKNAME" - Set nickname to be displayed to your chatpartner
  • !setinterests INTEREST1 "MULTIWORD INTEREST" ... - Set your list of interests
  • !setgender GENDER - Set your own gender.
  • !setimage IMAGELINK - Set a 'profile image' to be displayed to your peer. Must be direct HTTP(S) link to an image file.
  • !setdescription "DESCRIPTION" - Set a description to be displayed to your peer
  • !setnsfw - Toggles wether or not you want a NSFW chatpartner
  • !reset - Resets all settings

Social

  • !block - Block current chatpartner for 24h
